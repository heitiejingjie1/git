# 基础语法
## 标题
* "#"号的数量，代表几级标题

## 段落
* 空一行或多行，来分隔段落

## 换行
* 添加两个空格或使用<br>标签，来换行，推荐使用<br>标签

## 强调
### 粗体
* 在单词或短语两侧添加两个星号
* 例:  **这是我** 
### 斜体
* 在单词或短语两侧添加一个星号
* 例: *这不是我*
### 粗体＋斜体
* 在单词或短语两侧添加三个星号
* 例: ***这究竟是不我***
### 删除线
* 在单词或短语两侧添加两个波浪线
* 例: ~~这终究不是我~~
### 下划线
* 在单词或短语两侧添加两个下划线
* 例: __这也不是我__
### 上标
* 在单词或短语前侧添加一个上尖括号
* 例: 这是一个上标^2 
### 下标
* 在单词或短语前侧添加一个波浪线
* 例: 这是一个下标~2 

## 引用
### 段落引用
* 在段落前添加">"
> 把简单的事情考虑得很复杂，可以发现新领域；把复杂的现象看得很简单，可以发现新定律。-- 牛顿
### 块引用，多段引用
* 在段落前添加">"，每个段落前都要添加">"
> 这是第一段
> 这是第二段
> 这是第三段
### 嵌套引用
* 在嵌套引用前添加">>"
> 这是主引用
>> 这是嵌套引用 
### 引用还可包含其他元素，并非所有元素都支持嵌套，需要自己实践
>这是一个引用段落
> - 这是一个列表
> - 这是第二个列表
>>这是一个嵌套引用

3. c#
4. rust

## 代码
* 将其包裹在`````中
* `这是代码`
### 转义反引号
* 使用双重反引号
* ``这是`代码`的代码``
### 代码块
* 代码缩进至少四个空格或一个制表符
`
    <html>
      <head>
      </head>
    </html>
`
* 如果不想缩进使用围栏代码块
* 语法高亮在围栏代码块开头添加该语言名称
```c 
#include <stdio.h>
int main(){

  printf("hello world");
  return 0;
}
```

## 链接
* 链接名放在中括号中，链接地址放在括号中
* 这是[百度](http://www.baidu.com)
### 链接title
* 在地址后以""嵌入title
* 这是[bing](http://www.bing.com "必应官网")
### 网址和email
* 也可使用<>将网址和email变为地址
<http://www.bilibili.com>
<heitiejingjie@gmail.com>
### 强调链接
* 在链接前后增加星号来突出链接
### 引用类型链接
1. 第一部分:第一个方括号显示链接文本，第二个方括号显示标签
2. 第二部分:
    - 标签:以一组方括号包裹[table]:
    - 链接:以一组尖括号包裹"<>"
    - 链接文本:放在""或()中
    [1]: <http://www.baidu.com "百度">
